# Changelog

Все заметные изменения в проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-03-15

### Добавлено
- ✅ Поддержка двух AI-провайдеров: GigaChat и YandexGPT
- ✅ Переключение между провайдерами через inline-кнопки
- ✅ История диалогов с ограничением в 20 сообщений
- ✅ Команды: /start, /help, /clear, /provider, /models, /balance
- ✅ Клиент GigaChat с полной поддержкой API
- ✅ Клиент YandexGPT с поддержкой REST API
- ✅ Тестовые скрипты для обоих провайдеров
- ✅ Увеличенные таймауты (60 секунд) для стабильности подключения
- ✅ Форматирование ответов MarkdownV2
- ✅ Автоматическое обновление токенов доступа

### Изменено
- ⚙️ Обновлён python-telegram-bot до версии 22.6 (совместимость с Python 3.14)
- ⚙️ Обновлён gigachat до версии 0.2.0 (совместимость с Python 3.14)
- ⚙️ Улучшена обработка ошибок Unicode
- ⚙️ Оптимизирована работа с историей сообщений

### Исправлено
- 🐛 Исправлена совместимость с Python 3.14+
- 🐛 Устранены проблемы с кодировкой UTF-8 на Windows
- 🐛 Исправлено отображение названий моделей в GigaChat

### Технические детали
- Python 3.9+ требуется
- Виртуальное окружение обязательно
- Поддержка Windows, macOS, Linux
